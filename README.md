# Celiac Scanner

A cross-platform mobile application utilizing AI for real-time food label scanning to detect gluten, designed for individuals with Celiac disease.

## Project Vision
To provide a fast, reliable, and confidence-inspiring tool for real-time dietary restriction validation using the power of advanced AI models like Gemini 3 Flash. Built to scale with a robust CI/CD pipeline and monetization strategy.

## Tech Stack
- **Framework:** React Native
- **Language:** TypeScript
- **Backend & Services:** Firebase (Auth, Firestore, Cloud Functions, Hosting)
- **Monetization:** RevenueCat
- **AI Integration:** Nano Banana (Gemini 3 Flash)

## Setup Instructions
(To be updated as the environment is instantiated in code)
1. Clone the repository.
2. Ensure you have the React Native CLI setup correctly for iOS and Android.
3. Install dependencies: `npm install` or `yarn install`.
4. Run for iOS: `npx react-native run-ios`.
5. Run for Android: `npx react-native run-android`.

## Directory Structure
- `/ios` & `/android`: Native mobile codebases.
- `/functions`: Firebase Cloud Functions.
- `/assets`: Images, fonts, and branding materials.
- `/docs`: Architecture, prompt engineering versions, and task tracking.
